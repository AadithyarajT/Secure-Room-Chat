# SecureRoomChat

SecureRoomChat is a An, real-time chat application built using Django and WebSockets.

The platform prioritizes simplicity, performance, and user privacy.
 
Don't Just sell your priceless asset Information for just a simple chat with others

---

## Overview

SecureRoomChat enables users to communicate through secure chat rooms and direct messaging while maintaining strict privacy controls.  
Chats are temporary by design, and users can participate with or without registration.

---

## Features

- Real-time communication using WebSockets  
- Public and private chat rooms  
- Direct one-to-one messaging  
- Optional anonymous usage  
- Messages are not permanently stored  
- Clean and minimal user interface  
- Dark mode and light mode support  
- Anti-spam protection  

---

## Modules

- Admin Module  
- User Module (Web)  
- User Module (Mobile App)  

---

## User Capabilities

- Create and join public or private rooms  
- Participate in direct chats with friends 
- View and edit profile  
- Hide profile or username  
- Add and manage friends  
- View notifications  
- Send complaints or bug reports  
- Transfer room ownership  

---

## Admin Capabilities

- View and manage complaints and bug reports  
- Send replies and notifications  
- Create and manage global, regional, and special rooms.

---

## Privacy and Security

- No permanent storage of chat messages  
- Messages and media are removed when a room is closed or expires  
- No personal profile images (built-in avatars only)  
- Optional password-protected rooms  
- User identity can be hidden during chats  

---

## Technology Stack

- Backend: Django, Python  
- Real-Time Communication: Django Channels, WebSockets, Redis  
- Frontend (Web): HTML, CSS, JavaScript  
- Mobile Application: Flutter  
- Database: PostgreSQL  

---

## Project Goal

The goal of SecureRoomChat is to provide a secure, lightweight, and privacy-centric communication platform that works reliably across devices while maintaining a simple and intuitive user experience.


In simple Words:
"An platform providing temporary, user-initiated communication services without permanent data retention."
